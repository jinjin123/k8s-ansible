## ansible部署Kubernetes

系统可采用`Ubuntu 16.x`(未完成)与`CentOS 7.x`(建议7.x里使用最新的)
本次安装的版本：
> * Kubernetes v1.17.1 (HA高可用)
> * CNI plugins v0.8.1
> * Etcd v3.4.3
> * flanneld v0.11.0
> * Calico (不写,可以自行去找yaml部署)
> * Docker CE 19.03(可以19.06+,自行测试)
